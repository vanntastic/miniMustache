An example of how Mini Mustache works:

A simple example

Let's assume that you have the following hash: 

       var nums = {
          one: 1,
          two: 2,
          three: 3
        };
        
        // And you want to interpret the following block:
        Did you know that {one} + {two} = {three}?
Did you know that {{one}} + {{two}} = {{three}}?

Passing a collection

What if you wanted to pass a collection of hashes? Here's how that would work: 

      
      var collection = [{
        name: "James Doe",
        bday: "03/04/2010",
        description: "This is a quick description of james"
      },{
        name: "John Jackson",
        bday: "04/10/2001",
        description: "His name is John Jackson"
      }];
  • Name: {{name}}, Bday: {{bday}}, Description: {{description}}

A Remote Collection

Assuming you have a remote json file with an array of json objects. 

      // remoteCollection.js
      [{
        "name": "James Doe",
        "bday": "03/04/2010",
        "description": "This is a quick description of james"
      },{
        "name": "John Jackson",
        "bday": "04/10/2001",
        "description": "His name is John Jackson"
      }]
      
      // remote collection example
      $.getJSON("remoteCollection.json", function(data){
        $("#remote-list").miniMustache(data);
      });
  • Name: {{name}}, Bday: {{bday}}, Description: {{description}}
